# 浪漫旅程 ❤️

双人情侣棋盘冒险游戏 - 在线的真心话大冒险，增进情侣之间的感情。

## 功能特色

- 🎮 **实时双人游戏** - 基于 WebSocket 的实时同步
- 🎲 **趣味棋盘** - 掷骰子、触发事件、获得奖励
- 💕 **真心话大冒险** - 多种题库模式（普通/SM/异地恋/自定义）
- 🛒 **道具商店** - 遥控骰子、护盾、双倍引擎等道具
- ✨ **等级系统** - 经验值、升级解锁更多道具
- 🎭 **命运事件** - 随机触发的惊喜/挑战
- 🎨 **主题切换** - 多种配色主题
- 🎵 **背景音乐** - 浪漫背景音乐
- 🤖 **AI生成题目** - 支持多种 AI 模型生成自定义题目

## 快速开始

### 环境要求

- Node.js 18+

### 安装

```bash
git clone https://github.com/你的用户名/romantic-journey.git
cd romantic-journey
npm install
```

### 运行

```bash
npm run dev
```

访问 http://localhost:3000

### 生产构建

```bash
npm run build
npm start
```

## 技术栈

- **前端**: React 19 + TypeScript + TailwindCSS + Vite
- **后端**: Express + Socket.IO
- **AI**: 支持 DeepSeek、GLM、Qwen、Kimi、MiniMax
